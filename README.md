# 黄志文 - 个人简历网页

这是一个基于HTML/CSS/JavaScript开发的现代化个人简历展示网页。

## 📋 功能特点

- ✨ 现代化设计，响应式布局
- 🎨 渐变色彩方案，视觉效果出色
- 📱 完美支持移动端和桌面端
- 🖨️ 支持打印功能
- 🎭 流畅的动画效果
- ⌨️ 键盘快捷键支持
- 🔝 返回顶部按钮
- 💫 3D卡片悬停效果

## 🚀 使用方法

### 方法一：直接打开
1. 下载所有文件到同一目录
2. 双击 `index.html` 文件即可在浏览器中打开

### 方法二：本地服务器
```bash
# 使用Python启动简单服务器
python -m http.server 8000

# 或使用Node.js的http-server
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 🛠️ 自定义修改

### 修改个人信息
编辑 `index.html` 文件中的相应内容：
- 姓名、职位
- 联系方式
- 工作经历
- 项目经验
- 技能标签

### 修改配色方案
编辑 `style.css` 文件中的CSS变量：
```css
:root {
    --primary-color: #2563eb;    /* 主色调 */
    --secondary-color: #1e40af;  /* 次要色 */
    --accent-color: #3b82f6;     /* 强调色 */
}
```

### 添加新章节
在 `index.html` 中添加新的 `<section>` 元素：
```html
<section class="section">
    <h2 class="section-title">新章节标题</h2>
    <div class="content">
        <!-- 内容 -->
    </div>
</section>
```
